\begin{abstract}
\small
As the internal structure of companies network is relatively complex they have to adress special criteria for the securty of these networks. Escpecially as more and more companies are using VPN as part of ther network structure their security also has to be concerned as usually such VPNs are used to remotely connect to the companies network, providing additional attack surface.

Threfore any compromise of these VPNs is automatically a threat for the whole companies network itself. But this is exactly what occured with reagrds to CVE-2019-11510. Thus during this paper the vulnerability will be analysed, classified and rated. Additionally security recommendations for this vulnerability will be presented.

\end{abstract}
